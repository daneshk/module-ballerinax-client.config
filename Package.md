## Overview

The Client Config library contains common client configuration utilities and types for connectors.

### Key Features

- Common client configuration types shared across connectors
- Reusable utility functions for client initialization
- Standardized configuration patterns for connector development

### Compatibility
|                     | Version            |
|---------------------|--------------------|
| Ballerina Language  | Swan Lake 2201.2.0 |
